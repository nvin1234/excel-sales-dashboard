# excel-sales-dashboard
This project is an interactive Sales Dashboard built using Microsoft Excel. It provides insights into sales performance by analyzing data across different dimensions such as Salesperson, Region, and Product Items.  The dashboard uses Pivot Tables, Pivot Charts, and Slicers to allow dynamic filtering and easy data visualization.
Features
📈 Salesperson-wise sales performance analysis
🌍 Region-wise sales comparison (APAC, EMEA, LATAM, NA)
🛒 Product-wise filtering (Air Conditioner, Blender, Ceiling Fan, Coffee Grinder)
🎛️ Interactive slicers for real-time filtering
📊 Dynamic charts for better visualization

Tools & Technologies
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers


Dataset

The dataset used in this project contains:

Salesperson Name
Region
Product Item
Actual Price (Sales Value)


Key Insights
Identify top-performing salespersons
Compare sales performance across regions
Analyze which products generate the most revenue
Understand sales distribution visually
📌 How to Use
Download the .xlsx file from this repository
Open it using Microsoft Excel
Use slicers (filters) to explore data
Interact with charts to analyze insights
